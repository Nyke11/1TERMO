# 🖥️ Sistemas Operacionais & Segurança Cibernética: Guia de Aula

Este guia explora os fundamentos dos sistemas operacionais (Windows e Linux) e as práticas essenciais para proteger o ambiente digital. 🚀

---

## 1. 🐧 Sistemas Linux: O Poder do Código Aberto
O Linux é amplamente utilizado em servidores, supercomputadores e dispositivos IoT devido à sua estabilidade e segurança.

### 🛠️ Características Principais:
* **Kernel Monolítico:** O núcleo que gerencia a comunicação entre hardware e software. 🧠
* **Código Aberto:** Permite auditoria de segurança e personalização total. 🔓
* **O Terminal (CLI):** Uso intensivo de comandos para administração (ex: `ls`, `cd`, `chmod`, `chown`). ⌨️
* **Distribuições (Distros):** Variedades como Ubuntu (facilidade), CentOS (servidores) e Kali (segurança). 📦

---

## 2. 🪟 Sistemas Windows: Produtividade e Interface
Líder no mercado de desktops domésticos e corporativos, focado na experiência do usuário.

### 🏗️ Estrutura e Ferramentas:
* **Interface Gráfica (GUI):** Navegação intuitiva baseada em janelas e ícones. 🖱️
* **Registro do Windows:** Banco de dados que armazena configurações críticas do sistema. 🗃️
* **Gerenciador de Tarefas:** Monitorização em tempo real de CPU, RAM e disco. 📊
* **Active Directory (AD):** Serviço de diretório para gestão de identidades em redes empresariais. 🏢

---

## 3. 🛡️ Segurança Cibernética (Cybersecurity)
A proteção de sistemas e dados contra ataques digitais é uma prioridade em qualquer arquitetura de TI.

### ⚠️ Ameaças Comuns (Malware):
* **Vírus & Worms:** Programas que se replicam para danificar arquivos ou redes. 🦠
* **Ransomware:** Bloqueia o acesso aos dados e exige um resgate (geralmente em cripto). 💰
* **Phishing:** Engenharia social para roubar credenciais através de comunicações falsas. 🎣

### 🔒 Pilares da Proteção:
* **Firewall:** Barreira que filtra o tráfego de rede de entrada e saída. 🧱
* **Criptografia:** Garante a confidencialidade dos dados, tornando-os ilegíveis sem a chave. 🔐
* **MFA (Autenticação Multi-fator):** Adiciona uma camada extra de segurança além da senha. 🔑

## ⚖️ Comparativo: Windows vs. Linux

| Recurso | 🪟 Windows | 🐧 Linux |
| :--- | :--- | :--- |
| **Interface** | Predominantemente Gráfica | CLI e Diversas GUIs |
| **Custo** | Licenciamento Pago 💸 | Geralmente Gratuito 🆓 |
| **Segurança** | Alvo frequente de ataques | Permissões de ficheiros robustas |
| **Atualizações** | Muitas vezes forçadas | Controladas pelo utilizador |

---

## 🔗 O Fluxo de Gestão do SO
Um Sistema Operacional saudável e seguro opera nestas frentes:

1.  **Gestão de Recursos ⚙️:** Divisão justa de memória e processamento.
2.  **Controlo de Acesso 👤:** Definir quem pode ler, escrever ou executar ficheiros.
3.  **Monitorização 🕵️‍♂️:** Logs de eventos para detetar comportamentos suspeitos.
4.  **Backup 💾:** A última linha de defesa contra perda de dados.
