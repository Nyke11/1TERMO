🎓 Resumo Integrado: Tecnologia e Engenharia de Software
Este guia reúne os pilares fundamentais das nossas aulas: Infraestrutura (IoT e SO), Lógica de Programação e Engenharia de Requisitos.

1. 🌐 Arquitetura IoT & Hardware
A conexão entre o mundo físico e o digital acontece aqui.

🤖 Arduino (C++): Atua na Camada de Percepção. É o hardware que lê sensores e controla atuadores através de pinos I/O.

🐍 Python (Gateway): Atua na camada de inteligência, processando dados vindos do Arduino via Serial e enviando para a Nuvem ☁️.

💻 Código (Blink): Uso de setup() para configurar e loop() para executar.

2. 🖥️ Sistemas Operacionais & Segurança
A base onde todo software é executado.

🐧 Linux vs. 🪟 Windows: Linux é focado em código aberto e terminal (CLI); Windows foca em produtividade e interface gráfica (GUI).

🛡️ Segurança Cibernética: Proteção contra Malwares e Ransomwares.

🔒 Pilares: Uso de Firewalls, Criptografia e o princípio do privilégio mínimo.

3. 🧠 Lógica de Programação (Python)
Como estruturar o pensamento para resolver problemas.

📥 Entrada/Saída: input() para ler, int() para converter e print() para exibir.

🛣️ Decisões: Uso de if, elif e else para criar desvios no fluxo.

🔄 Repetições: while (enquanto durar a condição) e for (repetição contada).

✨ Clean Code: Escrever códigos com nomes claros e boa indentação para que outros humanos entendam. 🤝

📋 4. Engenharia de Requisitos
A fase de planejamento e definição do que será construído.

✅ Funcionais vs. 🛠️ Não Funcionais: O que o sistema faz vs. Como o sistema deve ser (seguro, rápido, estável).

🧠 Elicitação: Técnicas de Brainstorm e Entrevistas para extrair necessidades.

🎨 Validação: Uso de Prototipagem e Diagramas de Caso de Uso para alinhar expectativas com o cliente antes do código.

🔗 Fluxo de um Projeto Completo
Requisitos 📋: Define o que o cliente quer.

Lógica 🧠: Desenha o algoritmo da solução.

Desenvolvimento (IoT/SO) ⚙️: Escolhe o hardware e o sistema operacional base.

Segurança 🛡️: Garante que a solução é robusta contra ataques.
